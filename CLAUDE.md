# Shotliss Landing Site

## What is Shotliss

Pre-production filmmaking software covering storyboards, shot lists, and call sheets. Aimed at independent filmmakers, film students, and small studios — competing on simplicity and price, not features.

Tagline: **"Storyboards, shot lists, shooting made simple."**

## Site Purpose

A pre-launch marketing landing page. Goal: capture emails for early access, communicate the product clearly, and make people want it.

## Page Sections (in order)

1. **Hero** — tagline + main UI GIF (tab bar, storyboard, shot list loop)
2. **Feature Highlights** — three features, each with a short GIF:
   - Storyboards with built-in sketch tool
   - Smart shot lists with visual shot-type previews
   - Auto-generated call sheets from schedule
3. **How It Works** — longer workflow walkthrough GIF (storyboard → shot list → schedule → call sheet)
4. **Theme Demo** — light/dark + color accent toggle moment
5. **Pricing** — one-time ~$20 or monthly ~$3–5
6. **Free for Students** — verified student email = free access
7. **Email Capture** — "Early access pricing locks in forever. Get in before launch." + input + CTA

GIFs are placeholders for now — use `[GIF PLACEHOLDER — description]` comments or visible placeholder blocks in the UI.

## Design Direction

- Reference: Notion, Linear, Plane — clean, minimal, confident
- The site should feel like the product
- Light and dark mode supported (with color accent options), matching in-app theme toggle
- Single column, max ~800px wide on desktop, generous whitespace
- Nothing fights for attention

## Typography

- UI / Body: **Geist Sans**
- Headings: **Geist Sans heavy weight**
- Numbers, timecodes, shot IDs: **Geist Mono**

## Copy Principles

- Keep it terse and confident — no fluff
- Positioning: simplicity + price, not features
- "Everything an indie filmmaker actually needs, nothing they don't."
- Student section tone: "Film school is expensive enough."

## Tech

No framework decided yet — keep it simple. Static HTML/CSS/JS is fine unless there's a good reason not to.
